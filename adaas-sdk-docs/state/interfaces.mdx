# State Interfaces

This page documents the interfaces used in the `State` module for managing state between syncs.

## Grounding

- **Source File:** `src/state/state.interfaces.ts`

---

## `SdkState`

The core state managed by the SDK, which is combined with the connector-specific state.

<TypeTable
  type={{
    lastSyncStarted: { type: 'string', default: 'undefined' },
    lastSuccessfulSyncStarted: { type: 'string', default: 'undefined' },
    toDevRev: { type: 'ToDevRev', default: 'undefined' },
    fromDevRev: { type: 'FromDevRev', default: 'undefined' },
    snapInVersionId: { type: 'string', default: 'undefined' },
  }}
/>

---

## `AdapterState<ConnectorState>`

A generic type that combines the `SdkState` with a connector-specific state object.

```typescript
export type AdapterState<ConnectorState> = ConnectorState & SdkState;
```

---
