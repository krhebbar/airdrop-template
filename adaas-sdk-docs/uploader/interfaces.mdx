## Other Interfaces

- **`StreamAttachmentsResponse`**: The response from streaming multiple attachments.
- **`StreamResponse`**: The response from streaming a single attachment.
- **`StatsFileResponse`**: The response from a stats file operation.
- **`TransformerFileResponse`**: The response from a transformer file operation.
- **`ArtifactToUpload`**: Represents an artifact that is ready to be uploaded.
