## API Parameter and Response Interfaces

These interfaces define the structure for parameters and responses for the `Mappers` class methods.

- **`MappersGetByTargetIdParams`**: Parameters for the `getByTargetId` method.
- **`MappersGetByTargetIdResponse`**: Response from the `getByTargetId` method.
- **`MappersCreateParams`**: Parameters for the `create` method.
- **`MappersCreateResponse`**: Response from the `create` method.
- **`MappersUpdateParams`**: Parameters for the `update` method.
- **`MappersUpdateResponse`**: Response from the `update` method.
- **`UpdateSyncMapperRecordParams`**: A specific parameter structure for updating a sync mapper record.
