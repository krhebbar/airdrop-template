# Loading Types

This page documents the enums, interfaces, and type definitions used in the data loading process.

## Grounding

- **Source File:** `src/types/loading.ts`

---

## Enums

### `LoaderEventType`

Defines the types of events sent from the loader to the ADaaS platform to report progress and status.

### `ActionType`

Defines the possible actions that can be reported in a `LoaderReport`.
- `CREATED`
- `UPDATED`
- `SKIPPED`
- `DELETED`
- `FAILED`

## Function Types and Parameters

### `ExternalSystemLoadingFunction<Item>`

A generic function type for loading an item into DevRev.

```typescript
type ExternalSystemLoadingFunction<Item> =
  ({ item, mappers, event }: ExternalSystemItemLoadingParams<Item>)
  => Promise<ExternalSystemItemLoadingResponse>;
```

### `ExternalSystemItemLoadingParams<Type>`

Parameters for the `ExternalSystemLoadingFunction`.

### `ExternalSystemItemLoadingResponse`

The response from an `ExternalSystemLoadingFunction`.

### `ItemTypeToLoad`

Defines the loading functions for a specific item type.

## Other Types

- **`DonV2`**: A type alias for a string, representing a DevRev object reference.
- **`RateLimited`**: An interface for rate limit information.
- **`LoadItemResponse`**: The response from loading a single item.
- **`LoadItemTypesResponse`**: The overall response from loading multiple item types.
