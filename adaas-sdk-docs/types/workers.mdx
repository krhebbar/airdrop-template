# Worker Types

This page documents the enums, interfaces, and type definitions used for managing worker threads.

## Grounding

- **Source File:** `src/types/workers.ts`

---

## Enums

### `WorkerEvent`

Defines the standard events that can be emitted by a worker thread.
- `message`
- `online`
- `error`
- `exit`

### `WorkerMessageSubject`

Defines the subjects for messages passed between the main thread and a worker thread.
- `WorkerMessageEmitted`: For events emitted by the worker.
- `WorkerMessageExit`: To signal the worker to exit.
- `WorkerMessageLog`: For log messages from the worker.

## Other Interfaces

- **`SpawnInterface`**: Interface for the `Spawn` class.
- **`TaskAdapterInterface<ConnectorState>`**: Interface for the `TaskAdapter` class.
- **`WorkerMessage`**: A union type for all possible worker messages.
- **`GetWorkerPathInterface`**: Interface for getting the worker path.
