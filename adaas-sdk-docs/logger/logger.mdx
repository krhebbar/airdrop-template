# Logger Class

The `Logger` class is a custom logging utility that extends the built-in `node:console` module. It provides tagged logging and can be used in both the main thread and worker threads.

## Grounding

- **Source File:** `src/logger/logger.ts`

## Constructor

The constructor initializes a new `Logger` instance.

### Signature

```typescript
constructor({ event, options }: LoggerFactoryInterface)
```

## Usage Example

```typescript
import { Logger } from './logger';
import { AirdropEvent } from '../types/extraction';

// Create a mock AirdropEvent
const mockEvent: AirdropEvent = {
  // ... (populate with necessary event data)
};

// Create a new logger instance
const logger = new Logger({ event: mockEvent });

// Log messages
logger.info('This is an informational message.');
logger.warn('This is a warning message.');
logger.error('This is an error message.');
```
