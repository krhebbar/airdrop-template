## `PrintableArray`

An interface for representing a summarized array for logging purposes. This is useful for preventing overly verbose logs when dealing with large arrays.

<TypeTable
  type={{
    type: {
      type: 'string',
      description: 'The type of the printable object.',
      default: 'array',
    },
    length: {
      type: 'number',
      description: 'The number of items in the array.',
    },
    firstItem: {
      type: 'any',
      description: 'The first item in the array.',
      default: 'undefined',
    },
    lastItem: {
      type: 'any',
      description: 'The last item in the array.',
      default: 'undefined',
    },
  }}
/>

---

## `PrintableState`

An interface for representing a summarized object state for logging. This is used to provide a snapshot of an object's state without logging the entire object.

<TypeTable
  type={{
    '[key: string]': {
      type: 'any | PrintableArray | PrintableState',
      description: 'A key-value map representing the state of an object.',
    },
  }}
/>
